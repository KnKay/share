# share

This project is just another side project.
Due to several things in my mind I decided to make this thing a project.

Sharing stuff should be easy and fun. To make it more like a community I decided to give it an app.
The idea is based to a TV documentation about car sharing in a non urban part of germany.
I saw the need to make all the paper work in paper and manual. This should be more like a sharing thing now a days.

To be able to make it more than just car sharing I tend to keep it as open as possible.

## Dev

The dev should be an easy task using docker compose and VSCode. No real IDE needed for now.
The most information are, hopefully, in the doc folder.
