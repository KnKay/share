# Help

## What is used

* React
* queryclient
* axiosAPI

## React

React is the framwork used for we ui. This seem to be a good to use and easy to learn framework.
In general all pages are just a function the is tied to an entity. Typescript is not in use but will come in later.

React is returning an element that can be placed in the app.
Routers will take care of deciding which element is used

## queryclient and axios

Axios is used to take care of the login token and make all API calls.

query client is a framework that is taking care of state data and is an addon.

## Design framework
The design will be implemented using primereact. This is looking a bit more modern than bootstrap
